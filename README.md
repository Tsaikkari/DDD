# Daily Dose of Dopamine

This project is about raising one's happiness level.
https://daily-dose-of-dopamine.herokuapp.com/

## INSTALL

Dependencies:

- node >= 14
- mongod 4.x

```sh
$ yarn install
```

## RUN BACKEND

```sh
$ node server.js
```

## RUN FRONTEND

```sh
$ cd client
$ yarn install
$ yarn start
```
